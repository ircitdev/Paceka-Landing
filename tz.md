# Техническое задание: Лендинг для системы умных весов для пасеки

## Контекст проекта

Разработана инновационная система умных весов для пасеки с автоматическим учетом и мониторингом. Необходимо создать продающий лендинг для привлечения клиентов и продажи решения.

## Технический стек (рекомендуемый)

### Frontend

- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite 6.x
- **Styling:** TailwindCSS
- **Animations:** Framer Motion или GSAP
- **Forms:** React Hook Form + Zod
- **Icons:** React Icons или Lucide React
- **Image Optimization:** next/image или react-lazy-load-image

### Backend (опционально)

- **Option 1:** Python FastAPI (легкое API для форм)
- **Option 2:** Node.js + Express
- **База данных:** SQLite / PostgreSQL
- **Email:** Nodemailer / SendGrid

### DevOps

- **Hosting:** Vercel / Netlify (для frontend)
- **Backend Hosting:** Railway / Heroku / AWS
- **CI/CD:** GitHub Actions
- **Monitoring:** Google Analytics, Яндекс.Метрика

## Примерная структура лендинга (секции)

1. **Hero** - Главный экран с призывом к действию
2. **Проблема** - Боль целевой аудитории (ручное взвешивание, потеря данных)
3. **Решение** - Как наш продукт решает проблему
4. **Функции** - Детальное описание возможностей (6 блоков выше)
5. **Как работает** - Простая схема из 3-4 шагов
6. **Интерфейс** - Скриншоты и демо
7. **Преимущества** - Почему мы лучше традиционных методов
8. **Кейсы** - Истории успеха пасечников
9. **Тарифы** - Варианты комплектации и цены
10. **FAQ** - Ответы на частые вопросы
11. **CTA форма** - Финальный призыв к заказу
12. **Footer** - Контакты и юридическая информация
